About This Project

This project is designed to analyze text for specific keywords and detect potential risks using AI-powered analysis. It can process both typed and handwritten text from images using Optical Character Recognition (OCR). The goal is to provide accurate insights while maintaining data privacy.

AI-Powered Analysis

Image Recognition
